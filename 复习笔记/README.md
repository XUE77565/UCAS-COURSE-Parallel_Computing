1. **先读** [`00-复习要点与关键知识点.md`](00-复习要点与关键知识点.md) —— 总纲,含考点权重、必背公式、必会代码、概念对比表、按章复习要点。

## 考试速记三句话

- **MPI 手写代码是大头**(Bcast、点积、环形传递、Alltoall 矩阵转置、自定义类型处理矩阵)。
- **L03 可扩展性计算**(Amdahl $S=\frac{n}{1+(n-1)\alpha}$、Gustafson $S=n-(n-1)g$、等效率 $W=\frac{E}{1-E}T_o$)。
- **概念对比**(SIMD vs SIMT、强 vs 弱可扩展、假共享、阻塞 vs 非阻塞、单边 vs 双边)是高频小题。
